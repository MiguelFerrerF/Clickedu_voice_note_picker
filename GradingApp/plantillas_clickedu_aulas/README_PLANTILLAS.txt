========================================
FORMATO DE PLANTILLAS CLICKEDU
========================================

Para que el sistema clasifique y te muestre los nombres de las clases de forma amigable (Ej: "Biologia 1º BACH"), debes guardar los excels descargados de Clickedu en esta misma carpeta siguiendo una estructura de nombre concreta.

ESTRUCTURA RECOMENDADA (separada por guiones bajos):
materia_curso_grupo_nivel.xlsx

Ejemplos:
- biologia_1_ab_bach.xlsx     (Materia: Biología, Curso: 1, Grupo(s): A y B, Nivel: Bach)
- matematicas_3_a_eso.xlsx    (Materia: Matematicas, Curso: 3, Grupo: A, Nivel: ESO)
- ingles_2_b_eso.xlsx         (Materia: Ingles, Curso: 2, Grupo: B, Nivel: ESO)

Si usas nombres diferentes (ej: "notas_trimestre_1.xlsx"), el programa funcionará exactamente igual para las puntuaciones, pero el nombre mostrado visualmente en el menú desplegable de "Seleccionar Clase" no será tan estético.

Nota: El software no destruirá ni modificará nunca accidentalmente estas plantillas base; al presionar el botón de exportar se creará una copia nueva con las calificaciones inyectadas en una carpeta aparte del sistema llamada "notas".
